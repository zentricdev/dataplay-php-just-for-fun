# PHP Just For Fun

A place for coding PHP just for fun.

## Installation

Install via composer:

```bash
composer require dataplay/php-j4f
```

## Commands

| command           | description                    |
| ----------------- | ------------------------------ |
| `make help`       | List available _make_ commands |
| `make terminator` | Execute T-800 mission          |
| `make payday`     | Execute Pay Day example        |
